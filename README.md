# GProxy

#### 介绍
免费开源的谷歌浏览器插件，用于http代理浏览器搜索引擎。免去了开全局代理导致国内app无法使用或连接慢的情况。请勿用于商用。

#### 软件架构
软件架构说明
javascript + html

#### 安装教程

1.  下载此项目解压
2.  谷歌浏览器打开扩展程序加载解压好的文件夹即可。

#### 使用说明

1. 启用GProxy扩展，输入代理服务器的ip+端口即可
